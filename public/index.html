About tathya :
     Tathya is Soliton’s Digital Library initiative to maintain an internal knowledge database and make it accessible to every Soliton.Tathya is here to help you out and let other Solitons get benefited from your files. It is in turn getting an advantage of any Soliton’s time spent on identifying appropriate resources for one's own training, development and work related stuff.

Features
  1. Share resources with your collegues
  2. One click Webpage upload
  3. Save your precious time by avoiding useless webpages
Limitations
  1. Webpage with no title tags in its page source can be added to tathya,but cannot be searched
  2. Plugin is Supported only in chrome(Website is cross-browser independent)
  3. Plugin and Website requires independant authentication
  4. Works only in online
Future Works
  1. Developing plugin for mozilla and IE
  2. Improvised search results
  3. Upvote and Downvote to rate the webpages
  4. Extending the resources to pdf's,videos,images etc
How To Use Tathya

Here is a step by step instruction about how to use Tathya.

Sharing the webpage

Step 1 :Download and install tathya plugin to your chrome browser from chrome store.
Step 2 :When you feel the current web page as a useful,crispy and worth sharing, just click on the Tathya plugin which is displayed on the top right corner of your chrome browser.
Step 3 :Tathya plugin provides suggestions about the web pages from which you are allowed to select or add tags, along with your web page.Your web page is now sucessfully shared with others.

Searching the webpage

Step 1 :Go to Tathya website.Sign in using google account.
Step 2 :Search can be performed with tha help of textbox on the top-left of the webpage.
Step 3 :Results are displayed along with the google results.
Step 4 :Webpages uploaded by you can be viewed in Gallery page. Tags can be edited or URL can be removed from the same page itself.
Step 5 :User Activity is displayed in Activity log page.
Step 6 :Feed back can be posted in feedback page.
